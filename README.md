# Process Control Block & Ready Queue Implementation (C++)

## Overview
Implemented a Process Control Block (PCB) system and Ready Queue as part of an operating systems project. The system models process metadata and manages processes using a priority queue structure.

## Features
- PCB structure representing process metadata
- ReadyQueue implemented as a priority queue of processes
- Dynamic memory management
- Separation of interface and implementation (.h / .cpp)
- Robust error handling and memory-safe design

## Concepts Demonstrated
- Operating system process management
- Queue and priority queue data structures
- Object-oriented design in C++
- Modular system architecture
- Memory management and leak prevention
